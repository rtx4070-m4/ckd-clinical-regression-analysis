
# CKD GFR Regression Analysis

Project: Multivariate Regression Framework for Estimating GFR in Chronic Kidney Disease (CKD) Patients

Pipeline:
Raw Dataset → Cleaning → MAD Outlier Removal → Feature Selection → Model Development → Evaluation → Residual Analysis → Visualization

Models:
- Linear Regression
- LASSO Regression
- Ridge Regression
- Principal Component Regression (PCR)

Metrics:
- SSE
- R²
- AIC
